Sistemas de administración de base de datos (DBMS)

Es una colección de programas lo que permite a sus usuarios el acceso a una base de datos, manipular datos y ayuda en la representación de datos. Así mismo ayuda a controlar el acceso a la base de datos de varios por usuarios.

Los sistemas de gestión de bases de datos no son un concepto nuevo y fue implementado por primera vez en 1960. Charles Bachmen´s integrado del almacén de datos, se dice que es la primera en la historia en DBMS. Con el tiempo, las tecnologías de bases de datos evolucionaron mucho, mientras que el uso y las funcionalidades esperadas de las bases de datos se han incrementado enormemente.

Componentes de una DBMS

Es un conjunto de datos que se transforma en información relevante, este sistema consta de cuatro partes que son:

  • DBMS Jerárquico: Este tipo de DBMS emplea la relación de padre e hijo de almacenamiento de datos. Este tipo de DBMS es raramente usada en la actualidad, su estructura es como un árbol de nodos que representa los registros y ramas que representan los campos. El registro de Microsoft Windows utilizado en Windows 10 es un ejemplo de una base de datos jerárquica, opciones de configuración se almacenan como estructuras con árbol con nodos.
  • DBMS de red: Este tipo de DBMS admite relaciones de muchos a muchos, esto generalmente se traduce en complejas estructuras de base de datos. RDM server es un ejemplo de un sistema de gestión de bases de datos que implementa el modelo de red
  • DBMS relacional: Este tipo de DBMS define las relaciones de las bases de datos en forma de tablas también conocida como red de relaciones, a diferencia de los DBMS relacional no admite relaciones de muchos a muchos. DBMS relación suelen tener tipos de datos predefinidos que pueden apoyar, es el tipo de DBMS más populares en el mercado. Ejemplos de sistemas de gestión de bases de datos relaciones incluye Oracle, Microsoft SQL server.
  • DBMS relacionado a orientados a objetos: Este tipo admite el almacenamiento de nuevos tipos de datos los datos que se almacena tienen la forma de objetos. los objetos que se almacenaran en la base de datos tienen atributos, por ejemplo, género o edad, atributos para un objeto.

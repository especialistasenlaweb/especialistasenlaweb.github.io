Lenguaje HTML

HTML, siglas de Hyper Text Markup Language, es un lenguaje básico para crear cualquier página web. Es in lenguaje meramente descriptivo y se basa en etiquetas, no es un lenguaje basado en funciones, no es un lenguaje hiperactivo, es diferente a otros tipos de lenguajes como C y java. HTML simplemente quiere definir la estructura y el contenido, no va a permitir realizar aplicaciones, para eso, en cualquier página web si existen un tipo de aplicación y demás, es porque se basan en otro tipo de lenguaje que pueden ser complementarios a HTML. En conclusión, HTML no es el encargado de realizar esas funciones, sino que es el encargado de definir tanto la estructura como el contenido.

Tampoco permite gestionar la presentación de una página web, ya que HTML se apoya en las hojas de estilos las CSS y HTML es un lenguaje interpretado por los navegadores, de modo que no vamos a requerir ningún compilador de ningún intérprete para este tipo de lenguajes, sino que cualquier navegador que podamos encontrar en el mercado es capaz de interpretar el código HTML. Entre los navegadores q podemos encontrar en el mercado, encontramos Internet Explorer, Mozilla, Google Chrome, etc.

Hemos visto que para visualizar cualquier página web requerimos de un navegador y que requerimientos tiene HTML para crear esas páginas web? Pues con cualquier editor de texto podemos crear una página web, editores de textos hay muchos dependiendo del sistema operativo podemos encontrar diferentes, por ejemplo, el bloc de notas de Windows, pero un editor de texto sencillo de utilizar y gratis es notepad++ que se puede utilizar para diferentes plataformas.

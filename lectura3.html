Lenguaje PHP

PHP: (HYPERTEX PREPROCESSOR) es un lenguaje “open spource” interpretado de alto nivel embebido en páginas HTML y ejecutado en el servidor. Fue creado por Rasmus Lerdorf para uso personal en 1994. Es un lenguaje de programación muy potente que junto con HTML y sus múltiples librerías son usadas para hacer aplicaciones web y ejecutarla del lado servidor, esto quiere decir que con la ayuda de PHP nosotros podemos realizar aplicaciones web. Y que es una aplicación web? Es un sitio dinámico el cual existe la interacción entre un usuario y el sistema Algunas de las características de su enorme popularidad son: • Es un lenguaje de código abierto • Esta disponibles para muchos sistemas: GNU/Linux, Windows, Unix, etc. • Se puede usar con diversas bases de datos, pero especialmente con mysql y PostgreSQL • Al ejecutarse en el servidor, los programas PHP lo pueden usar todo tipo de máquinas con todo tipo de sistemas operativos Cabe aclarar que para ejecutar PHP debemos instalar o tener como servidor local o servidor en la nube el Apache y así poder ejecutar nuestro código PHP FUNCIÓN PRINCIPAL DE PHP Permitir la interacción de la página web con el visitante que pudo haber realizado cambio en ella y cada usuario que ingrese a la página podrá ver la información anteriormente modificada

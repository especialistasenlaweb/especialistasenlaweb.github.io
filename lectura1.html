PROGRAMACIÓN WEB

Hoy en día existen diferentes lenguajes de programación para desarrollar en la web, estos han ido surgiendo debido a las tendencias y necesidad de las plataformas. A medida que paso el tiempo, las tecnologías fueron desarrollándose, evolucionando y surgiendo nuevos problemas para la solución. Esto dio lugar al desarrollo del lenguaje de programación para la web, que permitiera interactuar con los usuarios y utilizar el sistema de base de datos.

Farzad Nazifi

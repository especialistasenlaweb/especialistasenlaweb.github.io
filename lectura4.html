Lenguaje ASP

Lenguaje ASP es una tecnología del lado servidor, desarrollada por Microsoft para el desarrollo de sitios web dinámicos. ASP significa Active server Pages, fue liberado por Microsoft en 1996. Las páginas web desarrolladas bajo este lenguaje es necesario tener instalado IIS. ASP no necesita ser compilado para ejecutarse, existen varios lenguajes que se pueden utilizar para crear paginas ASP, el más utilizado es VBScript, nativo de Microsoft. ASP se puede hacer también en PERL JS, no es java script, el código ASP puede ser insertado junto con el código HTML, los archivos cuentan con la extensión .ASP.

ASP Inicio hace 20 años específicamente en 1996. En 2002 aparece ASP.NET con el fin de poder desarrollar aplicaciones utilizando los lenguajes el framework.Net. En 2008 aparece asp.net MVC como respuesta a la aparición de framework de desarrollos modernos como Django, Spring.NET que utilizan el patrón modelo vista controlado. En 2010 aparece ASP.NET Web Pages y razor con un motor de vista. En 2012 aparece ASP:NET Web Api, para desarrollo de aplicaciones y SignalR para funcionalidades en tiempo real.

En 2014 aparece la nueva versión de Microsoft que es ASP.NET Core como respuesta a la web moderna. En la web moderna existe los Framework modernos y están orientado a móviles y tablets, utilizan diseño responsivo, es decir, se adapta la aplicación a la pantalla del dispositivo que se esté utilizando, se utiliza frameworks del lado del cliente como puede ser angularjs, están listo para implementarse en la nube y son multiplataforma, es decir, se pueden ejecutar en sistema operativo.
